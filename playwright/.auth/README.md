# Session Storage — Salesforce Authentication

This directory contains the Salesforce browser session file generated by the auth
setup script. This allows Task 1 tests to run without performing a full login on
every execution.

## Generating the Session
```bash
# Standard login (opens browser — complete MFA/email verification manually)
npm run auth:sf

## File Generated
| File | Description |
|---|---|
| `salesforce.json` | Salesforce org authenticated session |


## Session Expiry
Sessions expire after approximately **2 hours** of inactivity. The framework detects
expired sessions automatically and will open the browser for a fresh login when needed.
